# Legacy C4-16S2-E-B Audio Matrix Driver
This driver interfaces with this specific [Control4 audio matrix](https://www.control4.com/docs/product/audio-matrix-switch/data-sheet/english/latest/ "Control4 audio matrix"). Specifications on the matrix itself can be found on the Control4 site.

#### Properties

<a href="../../../../Assets/Knowledge-Base/Creator/Drivers/control4-audio-matrix-c4-16S2-e-b.png">
  <img src="../../../../Assets/Knowledge-Base/Creator/Drivers/control4-audio-matrix-c4-16S2-e-b.png" alt="Control4 Audio Matrix C4-16S2-E-B" width="300" height="">
</a>

* **Name:** Name of the device.

* **Location:** Location of the device within the Project. New Locations can be created by selecting this field, typing in a new name, and then selecting the corresponding "Add New Tag" option or pressing Enter on your keyboard.
<img src="../../../../Assets/Knowledge-Base/Creator/Drivers/locations-add.png" alt="Adding Main Dining Tag to Location" width="200" height="">

* **IP Address:** The destination IP address that SAVI will use when communicating with the device.
